# FUNDAMENTOS DE PROGRAMACIÓN EN JAVASCRIPT é Modulo4

## Objetivo del Proyecto

Este proyecto consiste en ser un repositorio de los diferentes ejercicios practicos del modulo 4

---

## Ejercicio practico 1

### Enunciado

En este Ejercicio Práctico, desarrollaremos un programa HTML simple que contiene la instrucción JavaScript para el método document.writeln(), situado dentro de un elemento <script>.

## Ejercicio practico 2
### Enunciado
En este Ejercicio Práctico, deberás realizar un programa que inserte el código JavaScript a partir de un archivo separado. En este código debes emplear el método window.alert() para devolver en el explorador un cuadro de alerta con la frase “¡Hola Mundo!”.
